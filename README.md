# openaiswitch_web

Marketing landing page for [openaiswitch](https://github.com/reigpol/openaiswitch) (`ais`).

## Local preview

```bash
python3 -m http.server 8765
# open http://127.0.0.1:8765/
```

## Deploy

Static site (`index.html`, `styles.css`, `main.js`, `assets/`).  
Host with any static host, or GitHub Pages from the `main` branch root.
